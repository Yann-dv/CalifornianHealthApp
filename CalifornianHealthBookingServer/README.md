# Californian Health Calendar Api

##  Description

This is the Booking Server for Californian Health. It use RabbitMq to communicate with the Booking Client.

## Installation

Download the project and build it with an IDE.

## Requirements

- .NET 7.0
- NuGet packages :
    > Microsoft.EntityFrameworkCore
    > Microsoft.EntityFrameworkCore.SqlServer
    > Newtonsoft.Json
- Libraries :
    > CalifornianHealthLib

## Usage

Run the project and use the Booking Client to communicate with it.

## Monitoring

You can monitor the server with the RabbitMq management interface, using the following credentials :
- Username : guest
- Password : guest
- Url : http://localhost:15672
